## Что делает эта программа

Эта программа представляет собой простой HTTP-сервер на базе Flask, который предназначен для получения и предоставления данных о VPN-подписке. Сервер делает запрос к указанному URL подписки, используя заголовки, имитирующие мобильное устройство (Android), и возвращает полученные данные в формате BASE64. Это позволяет использовать локальный сервер как прокси для VPN-клиентов, обеспечивая доступ к конфигурациям подписки без прямого обращения к внешнему источнику.

Главное преимущество которое даёт эта программа — это превышение лимита по устройствам в VPN сервисах. Программа автоматически генерирует случайные заголовки устройства (модель, HWID и т.д.) из списка поддерживаемых устройств, что позволяет обходить ограничения на количество подключений. Конфигурация устройства хранится в файле config.yaml, который создается автоматически при первом запуске.

## Шаги к установке:
- Создайте виртуальное окружение — `python -m venv .venv`
- Скачайте все зависимости — `pip install -r requirements.txt`
- Запустите main.py файл — `python main.py`
- При первом запуске автоматически создадутся файлы .env и config.yaml в корне проекта.
- Откройте файл .env и измените значение "change_me" на URL вашей подписки.
- Файл config.yaml содержит случайные заголовки устройства; вы можете отредактировать его, если необходимо (например, для использования конкретной модели устройства).

## Запуск и использование:
- Запустите main.py файл — `python main.py`
- В браузере попробуйте зайти на localhost:5100
- Если видите данные конфигурации серверов в кодировке BASE64, значит всё прошло успешно.
- Добавьте локальный адрес с портом устройства, на котором запущен этот http-сервер, в VPN-клиент целевого устройства. (Например, http://192.168.1.58:5100/)
